This dir contains the major inputs and outputs across the entire analysis 

The number proceeding the file name cooresponds to the step it was used in/made from. (Remove if running script)

The total_lineage_info is the output of all viral lineage information from Step 6 (Viral taxonomy). The VMR spreadsheet is the download used to annotate the genetic composition of the viruses

The Virome_read_count_info contains the read counts from steps 0-7 of the virome analysis.
